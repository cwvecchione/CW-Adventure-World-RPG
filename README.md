# CW-Adventure-World-RPG
A simple, adventure-style RPG in a robust world, built using Phaser.js.  Contains a large map for players to explore, monsters to fight, and gold to collect. 

## Running the Game
The easist way to run the game is to download the repo, open a command-line terminal in that directory, and run the command: npx http-server

## Planned Features
Updated Graphics, More Monsters, Weapon Power-Ups, Muliplayer
